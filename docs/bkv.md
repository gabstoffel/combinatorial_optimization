# Melhores Valores Conhecidos (BKV)

**BKV** (do inglês *best known value*) são os melhores valores conhecidos da função objetivo
para cada instância. Servem de referência para medir a qualidade das soluções obtidas — tanto
pelo solver quanto pela meta-heurística — por meio do desvio percentual em relação a esses valores.
